## Introduction

Python is a high-level, interpreted, general-purpose programming language known for its simplicity, readability, and flexibility.  
It allows developers to write clean and efficient code for small scripts as well as large enterprise applications.

Python supports multiple programming paradigms such as procedural, object-oriented, and functional programming.

**Q1. What is Python mainly used for?**

Yes, Python is one of the most beginner-friendly languages.

**Q3. Is Python free?**

Yes, Python is widely used in enterprise and production environments.
